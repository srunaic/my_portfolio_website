myPortfolio
게임 & 풀스택 개발자를 위한 현대적이고 인터랙티브한 포트폴리오 웹사이트입니다.
GitHub 프로젝트, 문서, 보유 기술을 프리미엄 애니메이션과 다크 테마로 시각화하여 보여줍니다.

주요 기능 (Features)
* 히어로 섹션 (Hero Section)
3D 스타일의 파티클 배경(시뮬레이션)과 자기소개 영역
* 기술 스택 (Skills)
Unity, Server, Web 영역별로 분류된 배지 형태의 기술 목록
* 프로젝트 갤러리 (Project Gallery)
GitHub 및 Google Drive 문서로 연결되는 인터랙티브 카드 UI
* 사용 기술 (Tech Stack)
React 19, TypeScript, Tailwind CSS v4, Framer Motion

로컬 실행 방법 (How to Run Locally) 
1. 이 폴더를 VS Code 또는 터미널에서 엽니다.
2. 의존성 패키지를 설치합니다 (이미 설치되어 있다면 생략 가능).
3. npm install
4. 개발 서버를 실행합니다.
5. npm run dev
6. 터미널에 표시되는 주소 (예: http://localhost:5173)를 브라우저에서 엽니다.

프로덕션 빌드 방법 (How to Build for Production)
정적 빌드 파일을 생성하여 서버에 업로드하려면 아래 명령어를 실행합니다.
npm run build
빌드 결과물은 dist 폴더에 생성됩니다.
